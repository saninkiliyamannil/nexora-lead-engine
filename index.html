Backend server URL
On Netlify, this can stay as the same site URL. The app defaults to the current site domain and calls serverless functions under `/api`.
How to deploy on Netlify
1. Push this project to GitHub
2. Import the repo in netlify.com
3. Netlify will publish the frontend and route `/api/*` to serverless functions
4. Add environment variables: GROQ_API_KEY and RAPIDAPI_KEY
5. Deploy the site and use the Netlify URL above only if you want to override auto-detection
